Games


Foul Machinations

Foul Machinations is a game currently in development and was my game development portfolio project submission for third year.

It was developed using Unreal Engine 5 and is is aimed to be a Souls-like RPG with inspirations from JRPGs such as Octopath Traveler


River Catcher

Link to the latest build of the game:

Link

River Catcher is a team collaborative project between students at the University of the West of Scotland.

It is being developed using Unity and the target platform is mobile. The aim of the game is to clean up trash from a dirty river, then using the points you gain from collecing them to upgrade your boat, leading to a semi-afk experience.


Mega Mecha Fighter

Mega Mecha Fighter is a fighting game where you control mechanised robots to fight against enemy mechanised robots

It is being developed in BabylonJS. The game revolves around the main character Rudd, who is undergoing simulations to help fight the war against the Fernweh.


The Anomaly

The Anomaly is a production between Mark Clyne and Matthew Baldwin for JavaScript Games in Year 4 of Computer Games Development.

It is being developed in BabylonJS and is a fantasy science-fiction game where you must escape the Anomaly at all costs.


Proto Discovery (Name TBD)

Proto Discovery is a mystery adventure game where you discover what is going on, on this specific week every year.

It is in development using Unreal Engine 5.0.1 and it is aimed to be a narrative driven game where players make choices based on information they learned earlier in the game.
